# Three.js Demo: Chair Variants + Click Popup

Этот пример показывает две ключевые механики для заказчика:

1. Переключение вариантов предмета в одной точке сцены (по уровню).
2. Клик по 3D-объекту и открытие попапа.

## Как запустить

1. Открой проект в папке `c:\projects\example`.
2. Запусти любой статический сервер из корня проекта.
3. Открой `index.html` в браузере через `http://localhost:...`.

Пример сервера через Python:

```bash
python -m http.server 8080
```

## Подключение твоей модели

Положи файл модели в:

`assets/cafe.glb`

Если файл не найден, включится fallback-сцена.

## Как назвать объекты в Blender

Для автоматического переключения вариантам нужны имена:

- `Chair_1`
- `Chair_2`
- `Chair_3`

Поддерживаются также варианты:

- `Chair_V1`, `Chair_V2`, `Chair_V3`
- `Chair_Level1`, `Chair_Level2`, `Chair_Level3`

Рекомендация:

- Сделай каждый вариант отдельным parent/group (внутри может быть сколько угодно мешей).
- Поставь все 3 варианта в одну и ту же позицию.
- Экспортируй в `.glb`.

## Что уже реализовано в коде

- По кнопкам "Уровень 1/2/3" показывается только один chair-вариант.
- По наведению на стул рисуется `BoxHelper`.
- По клику на стул открывается попап (имитация бизнес-ивента).
